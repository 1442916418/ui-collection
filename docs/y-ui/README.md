# y-ui

> :art: 面向未来的原生 Web Components + 拟物风格组件库

* 拟物风格 - **light** 或 **dark** 风格 **UI** 样式。
* 跨框架 - 无论是 **react**、**vue** 还是原生项目均可使用。
* 组件化 - **shadow dom** 真正意义上实现了样式和功能的组件化。
* 类原生 - 一个组件就像使用一个 **div** 标签一样。
* 无障碍 - 支持键盘访问。
